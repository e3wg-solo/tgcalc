# Деплой на Amvera

Конфигурация проекта уже настроена для развертывания на [Amvera Cloud](https://docs.amvera.ru/).

## Что сделано

- Добавлен файл `amvera.yml` с окружением **Node.JS Browser**
- Указан путь к артефактам Vite: `dist/*`
- Используется Node.js 18

## Шаги для деплоя

### 1. Создать проект в Amvera

1. Откройте [личный кабинет Amvera](https://cloud.amvera.ru/)
2. Нажмите **«Создать проект»**
3. Выберите тип сервиса: **«Приложение»**
4. Укажите название (например, `tgcalc`), тарифный план и нажмите **«Далее»**

### 2. Подключить GitHub-репозиторий

1. В разделе **«Загрузка данных»** выберите **«Загрузка через git»**
2. Подключите GitHub и выберите репозиторий `e3wg-solo/tgcalc`
   - Инструкция: [Подключить GitHub, GitLab, Bitbucket](https://docs.amvera.ru/applications/git/webhooks.html)
3. Убедитесь, что ветка — `main` (или `master`, если используется она)

### 3. Конфигурация

Файл `amvera.yml` уже в репозитории. Amvera использует окружение **Node.JS Browser** и артефакты из `dist/`.

Если в интерфейсе Amvera есть шаг настройки окружения — выберите **Node.JS → Browser**.

### 4. Сборка и запуск

После `git push` в ветку `main` Amvera:

1. Выполнит `npm install && npm run build`
2. Скопирует файлы из `dist/` в корень приложения
3. Запустит nginx для раздачи статики с поддержкой SPA-маршрутизации

### 5. Проверка

После успешного деплоя приложение будет доступно по ссылке из личного кабинета Amvera.
